Projects

Crowdsourced Flood Mapping Web Application

Aim to track the real-time impact of floods on transportation networks using flood-related hashtags in Ghana. Provide real-time map updates of flood-prone areas, during extreme weather events. Implement sentiment analysis from social media data to understand public perceptions of disruptions due to flooding.

Evaluation of the Impact of Activities in the Sisa Catchment on Sisa Stream

Employed HEC-RAS to model stream flow and analyze flood risks. Conducted land-cover analysis using GIS to map and assess the impact of land-use changes on the stream. Identified significant changes in land use patterns contributing to increased flood risks.
